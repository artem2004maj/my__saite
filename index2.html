JSON в Python

Использование и применение JSON в Python

JSON (JavaScript Object Notation) - это легковесный формат обмена данными, который легко читается и пишется как людьми, так и машинами. Python имеет встроенную поддержку работы с JSON через модуль json.

Основные функции модуля json:

  • 'json.dumps(obj)': Сериализует объект Python в строку JSON.
  • 'json.loads(s)': Десериализует строку JSON в объект Python.
  • 'json.dump(obj, fp)': Сериализует объект Python в файл в формате JSON.
  • 'json.load(fp)': Десериализует файл JSON в объект Python.

Примеры кода:

1. Сериализация и десериализация строк:


import json

# Сериализация объекта Python в строку JSON
data = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
                        
json_string = json.dumps(data)
print(json_string)  # Вывод: {"name": "John", "age": 30, "city": "New York"}
                        
# Десериализация строки JSON в объект Python
parsed_data = json.loads(json_string)
print(parsed_data)  # Вывод: {'name': 'John', 'age': 30, 'city': 'New York'}

2. Работа с файлами:


import json

# Сериализация объекта Python в файл JSON
data = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

with open("data.json", "w") as file:
    json.dump(data, file)

# Десериализация файла JSON в объект Python
with open("data.json", "r") as file:
    loaded_data = json.load(file)

print(loaded_data)  # Вывод: {'name': 'John', 'age': 30, 'city': 'New York'}

2. Работа с файлами:


import json

# Объект Python с вложенными словарями и списками
data = {
    "name": "John",
    "age": 30,
    "city": "New York",
    "hobbies": ["reading", "traveling", "sports"],
    "address": {
        "street": "123 Main St",
        "zipcode": "10001"
        }
}
                        
# Сериализация объекта Python в строку JSON
json_string = json.dumps(data, indent=4)
print(json_string)
                        
# Вывод:
# {
#     "name": "John",
#     "age": 30,
#     "city": "New York",
#     "hobbies": [
#         "reading",
#         "traveling",
#         "sports"
#     ],
#     "address": {
#         "street": "123 Main St",
#         "zipcode": "10001"
#     }
# }

Применение JSON в Python:

  • Обмен данными между клиентом и сервером: JSON широко используется для передачи данных между веб-приложениями и серверами.
  • Конфигурационные файлы: JSON можно использовать для хранения настроек приложений.
  • Сериализация данных: JSON удобен для сериализации сложных объектов Python.
  • Кэширование данных: JSON можно использовать для хранения временных данных в файлах.
